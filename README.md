# SoniX

A **SwiftUI music player UI prototype** — inspired by modern music apps and built as a **school project**.

> SoniX is a front-end–only app created to explore SwiftUI layout, state management, and interface design.  
> It is **not a real music streaming service**.

##  About

soniX is a SwiftUI-based app that mimics parts of a music player experience, such as browsing songs and viewing playback screens.

The focus of this project is **interface design**, not real audio playback or streaming infrastructure.

---

##  Features

Current functionality includes:

-  Song list browsing
-  Mini player bar UI
-  Full player screen with artwork and metadata
-  Modular SwiftUI components
-  Responsive layouts using SwiftUI

---

##  Project Structure

Key files in the repository:

- `SoniXApp.swift` — App entry point  
- `ContentView.swift` — Root navigation view  
- `Song.swift` — Song data model  
- `SongListCard.swift` — Song list UI component  
- `MiniPlayerBar.swift` — Bottom mini-player UI  
- `FullPlayerView.swift` — Expanded player interface  
- `SongDetailView.swift` — Track details and artwork  
- `RecentlyPlayedLoaderView.swift` — Simulated recent activity view  

All UI is built using **SwiftUI**.
